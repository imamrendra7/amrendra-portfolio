# 🚀 Amrendra's Developer Portfolio

Welcome to my personal developer portfolio website! Built using modern web technologies to showcase my skills, achievements, experience, and projects.

🔗 **Live Link:** [https://imamrendra7.github.io/amrendra-portfolio/](https://imamrendra7.github.io/amrendra-portfolio/)

---

## 🌟 Key Features
- 💻 **Interactive Developer Terminal:** A fully functional mock terminal simulating command-line actions (DSA, Projects, Resume download, and details).
- 🌓 **Dark & Light Mode:** Seamless toggle between dark and light themes.
- 📬 **Interactive Contact Form:** Integrated with EmailJS for direct inquiries.
- 📈 **Animated Sections:** Dynamic scrolling effects powered by Framer Motion.
- 📱 **Fully Responsive Layout:** Optimized for mobile, tablet, and desktop screens.
- 📄 **Direct Resume Download:** One-click CV download link.

---

## 🛠️ Built With
- **Frontend Core:** [React.js](https://react.dev/) & [TypeScript](https://www.typescriptlang.org/)
- **Build Tool:** [Vite](https://vite.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/) & [React Scroll](https://www.npmjs.com/package/react-scroll)
- **Contact Form:** [EmailJS](https://www.emailjs.com/)

## 🚀 How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/imamrendra7/amrendra-portfolio.git
   cd amrendra-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the local development server:**
   ```bash
   npm run dev
   ```

4. **Build the production bundle:**
   ```bash
   npm run build
   ```

---

## 📦 Deploying to GitHub Pages

This project is configured to deploy automatically to GitHub Pages. To publish changes:
```bash
npm run deploy
```
